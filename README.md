## 🚩简介

- 跨平台Markdown编辑器（Mac、Windows、Linux）
- 多种主题和代码块样式
- 支持数学公式（行内公式和块公式）
- 支持UML图，参考 <http://www.plantuml.com/>
- 支持自定义图片大小（用法：\![](123.png =100x100)）
- 通过{}自定义HTML标签的属性或CSS样式
- 支持多种写作字体，可调节字体大小，支持夜间模式
